## ezEdits이란 무엇인가요?

ezEdits는 수많은 새로운 명령, 패턴, 마스크, 도구를 추가하여 놀라운 창작물을 만드는 데 도움이 되는 WorldEdit/FastAsyncWorldEdit(WE/FAWE) 확장 플러그인입니다!
